Google Sheets via Apps Script

Belum terhubung
1

Buat Google Apps Script

Deploy script sebagai Web App untuk menghubungkan SchoolPlanner ke spreadsheet kamu.

①Buka script.google.com → klik New Project
②Hapus kode default, paste kode dari tombol di bawah. Ganti SPREADSHEET_ID
③Klik Deploy → New Deployment → Web App
④Set Execute as: Me dan Who has access: Anyone → Deploy
⑤Copy URL Web App → paste di Step 2
2

Hubungkan ke SchoolPlanner

Paste URL Web App dari Apps Script kamu. URL otomatis tersimpan.

3

Sinkronisasi Data

Auto-refresh: off

Tarik data dari spreadsheet atau push semua tugas ke spreadsheet.

Ambil tugas dari spreadsheet → tambahkan ke app (skip duplikat)

Kirim semua tugas dari app → tulis ke spreadsheet (overwrite)
